To run the live app

1. Install Node.js if they don't have it — https://nodejs.org/, click LTS, run installer, accept defaults.
2. In a terminal at the repo root:
   npm install
   npm run dev
3. Open the printed URL. On a phone on the same Wi-Fi you can also use the "Network:" URL for a real phone-shaped view (this app is mobile-first).
